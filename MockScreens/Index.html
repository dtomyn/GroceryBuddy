About

Grocery Buddy is an application developed by the Solvera UltraHackathon Team comprised of: Darek Tomyn, Tom Yarnton and Jayper Sanchez

The application attempts to show the following features:

  • jQuery mobile
  • Knockout.js
  • Storage
  • Offline
  • Web API
  • PhoneGap
  • Camera / barcode
